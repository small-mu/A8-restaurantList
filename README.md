# AC_RestaurantList

## 安裝與執行步驟 (以下為終端機輸入語法)
* 下載檔案  
  `git clone https://github.com/small-mu/A8-restaurantList.git`
 
* 切換至該資料夾  
 `cd AC_RestaurantList`

* 安裝expres等上述環境建置  
 `npm install express`

* 執行程式  
 `npm run dev`

* 下載種子資料
 `node models/seeds/restaurantSeeder.js`

* 啟動完成請至 http://localhost:3000/
